# Mage React Cam Demo

This project is a practical demonstration of the [mage-react-cam](https://www.npmjs.com/package/mage-react-cam) library. It showcases how to implement a camera interface in React with support for snapshots, digital zoom, and camera switching.

## 🚀 Overview

The demo provides a simple interface to:
- **Live Stream**: View the camera stream in real-time.
- **Capture Snapshots**: Take high-quality JPEG snapshots from the video feed.
- **Control Zoom**: Test the digital zoom capabilities (Zoom In/Out).
- **Switch Cameras**: Toggle between the front (user) and back (environment) cameras.
- **Monitor Levels**: View the current and maximum supported zoom levels for your device.

## 🛠️ Installation & Running

To run this demo locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/teilorbarcelos/mage-react-cam-demo.git
   cd mage-react-cam-demo
   ```

2. **Install dependencies**:
   ```bash
   yarn install
   ```

3. **Run the development server**:
   ```bash
   yarn dev
   ```

4. **Open in browser**:
   Navigate to `http://localhost:3000` to see the demo in action.

## 📦 About the Library

`mage-react-cam` is a lightweight React component that abstracts the `MediaDevices` API, making it easy to integrate camera features with minimal boilerplate.

**Main features:**
- 📸 Base64 Snapshots
- 🔍 Digital Zoom Control
- 🔄 Facing Mode Switching
- ⚡ TypeScript Support
- 🔋 Low Overhead

[Check out the full documentation here](https://github.com/teilorbarcelos/mage-react-cam)
